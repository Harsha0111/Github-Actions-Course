-------------------

cd react-netlify-demo
npx create-react-app front-end
cd front-end


Create a Flask app in another folder:
------------------------------------
mkdir flask-render-demo
cd flask-render-demo
python3 -m venv venv
source venv/bin/activate
pip install flask gunicorn


Log into Netlify:
-----------------
Go to Netlify and sign in or create an account.
Click on “New site from Git” to link your repository with Netlify.
Manually deploy the app by selecting the repository and branch (e.g., main).
After the manual deployment, note the Site ID from the settings (you'll use this in the GitHub Actions workflow).


Set Up GitHub Secrets for Netlify:
----------------------------------
Add your Netlify credentials in your GitHub repository:
Go to your repository’s Settings > Secrets and variables > Actions.
Click on New repository secret.
Add the following secrets:
NETLIFY_AUTH_TOKEN: You can generate this from your Netlify account settings under “Personal Access Tokens.”
NETLIFY_SITE_ID: The Site ID you got from the Netlify dashboard.


name: 03.Deploy to Netlify and Render

on:
  push:
    branches:
      - main
  pull_request:
    branches:
      - main

jobs:
  # Deploy React Frontend to Netlify
  deploy-react:
    runs-on: ubuntu-latest

    steps:
      - name: Checkout code
        uses: actions/checkout@v3  # Checkout your repository

      - name: Set up Node.js
        uses: actions/setup-node@v3
        with:
          node-version: '16.x'  # Specify the Node.js version

      - name: Install dependencies
        run: npm install
        working-directory: modules/03.Continuous_Delivery/front-end

      - name: Build the React app
        run: npm run build
        working-directory: modules/03.Continuous_Delivery/front-end

      - name: Deploy to Netlify
        uses: netlify/actions/cli@master
        with:
          args: deploy --dir=modules/03.Continuous_Delivery/front-end/build --prod  # Deploy to production
        env:
          NETLIFY_SITE_ID: ${{ secrets.NETLIFY_SITE_ID }}  # Use the Site ID secret
          NETLIFY_AUTH_TOKEN: ${{ secrets.NETLIFY_AUTH_TOKEN }}  # Use the Auth Token secret
